# Gestor de Fútbol

## Descripción
Módulo de Odoo para la gestión integral de clubes de fútbol. Este módulo permite administrar:
- Clubes de fútbol
- Jugadores y sus estadísticas
- Entrenadores
- Competiciones y partidos

## Instalación
1. Clona este repositorio en tu directorio de addons de Odoo.
2. Actualiza la lista de aplicaciones en Odoo.
3. Busca "Gestor de Fútbol" e instálalo.
